# 📊 SCT_DA_3 - Interactive Visualization Dashboard

## 📌 Project Overview

This project was completed as part of the SkillCraft Technology Data Analytics Internship. The objective was to create an interactive dashboard using Power BI to analyze employee attrition and generate actionable insights.

## 🎯 Objectives

* Build an interactive Power BI dashboard
* Analyze employee attrition trends
* Identify factors affecting employee retention
* Enable dynamic filtering and data exploration

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization

## 📈 Dashboard Features

* Attrition by Department
* Attrition by Job Role
* Attrition by Age Group
* Attrition by Job Satisfaction
* Attrition by Overtime
* Interactive Filters and KPIs
